# discordbot

基本功能
游戏与频道消息互相转发

管理频道发送help/帮助  获取命令帮助
管理员基本维护命令

> 玩家频道显示当前在线人数，服务器加入人数，服务器在线时间
管理员频道显示服务器负载

-配置文件需要：  
  服务器id  
  服务器管理频道id  
  Chat信息转发频道id  
  Bot Token  

备份:   
. 灵活的Q群消息触发备份 还有Cron计划备份 修改shibot.json配置文件 设置Cron触发时间
